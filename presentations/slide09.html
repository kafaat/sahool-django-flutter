الخدمات المصغرة بالتفصيل

الخدمات الأساسية
Auth Service
Node.js + TypeScript
تسجيل المستخدمين، تسجيل الدخول، إصدار JWT tokens، إدارة الصلاحيات
User Service
Go + Gin
إدارة بيانات المستخدمين، المزارع، الحقول، الملفات الشخصية
Notification Service
Node.js + TypeScript
إرسال إشعارات (Push, Email, SMS)، إدارة تفضيلات الإشعارات
الخدمات الذكية
GeoChat Service
Python + FastAPI
معالجة اللغة الطبيعية، تنسيق الاستجابات، إدارة السياق
Image Processor
Python + Flask
معالجة الصور، تطبيق نماذج AI/ML، حساب مؤشرات النباتات
Recommendation Service
Python + FastAPI
توليد توصيات الري، الأسمدة، مكافحة الآفات
خدمات الاستشعار
Sentinel Service
Python + FastAPI
التكامل مع Sentinel Hub API، جلب الصور الفضائية
FIBOA Service
Python + Flask
تشغيل نموذج FIBOA، كشف حدود الحقول
Temporal Analysis
Python + FastAPI
تحليل السلاسل الزمنية، كشف الاتجاهات والشذوذات
خدمات الزراعة
Crop Detection
Python + FastAPI
كشف نوع المحصول، مرحلة النمو، تقييم الصحة، كشف الأمراض
Nitrogen Service
Python + Flask
تقدير مستويات النيتروجين، توصيات التسميد النيتروجيني
Weather Service
Python + FastAPI
جلب بيانات الطقس، حساب مؤشرات زراعية مناخية
IoT Gateway
Node.js + TypeScript
استقبال بيانات MQTT، تخزين قراءات المستشعرات، إرسال أوامر
مبادئ التصميم والتواصل
استقلالية الخدمات
كل خدمة مستقلة تماماً ويمكن تطويرها بشكل منفصل
التواصل عبر APIs
HTTP/gRPC للطلبات الفورية، Message Queue للمهام الطويلة
قابلية التوسع الأفقي
يمكن توسيع كل خدمة أفقياً حسب الحاجة
مرونة الفشل
فشل خدمة واحدة لا يؤدي إلى توقف النظام بأكمله
