استراتيجية قواعد البيانات
تتبنى المنصة استراتيجية Database per Service، حيث تمتلك كل خدمة مصغرة قاعدة بياناتها الخاصة أو schema منفصل. هذا يضمن استقلالية الخدمات ويمنع الاعتماد المباشر على بيانات خدمات أخرى.
PostgreSQL
قاعدة بيانات علائقية قوية للبيانات المنظمة والمعاملات المعقدة
PostGIS
امتداد جغرافي لـ PostgreSQL للبيانات المكانية والخرائط
InfluxDB
قاعدة بيانات مخصصة لسلاسل الوقت وقراءات المستشعرات
Redis
مخزن مؤقت في الذاكرة لتحسين الأداء وتقليل الاستجابة
AWS S3
تخزين كائنات للصور الفضائية والملفات الكبيرة
Elasticsearch
محرك بحث قوي للبيانات النصية والتحليلات
